## WHY

Some Bro don't want to be blacklisted for running small memory tasks on large memory nodes.

## INSTALL

```bash
cargo install --git https://github.com/wjwei-handsome/memonster
```

## USAGE

```bash
memonster --help
memonster -m 10G
memonster -m 250Gb
```
